Sudoku
------

A highly efficient sudoku-solving program, open sourced under MIT license.

### Input Sample

> 0 0 0 4 0 0 8 7 3
> 0 4 0 0 5 0 0 0 0
> 8 0 0 2 1 0 0 0 0
> 0 2 4 0 0 0 0 0 0
> 3 0 5 0 0 0 7 0 9
> 0 0 0 0 0 0 5 3 0
> 0 0 0 0 4 9 0 0 1
> 0 0 0 0 6 0 0 8 0
> 5 1 3 0 0 2 0 0 0

(0 stands for unknown, or to be filled)
